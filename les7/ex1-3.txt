ex 1
1. sudo yum install rabbitmq-server
2. sudo rabbitmq-plugins enable rabbitmq_management
3. sudo service rabbitmq-server restart
4. настраиваем проброс портов для виртуальной машины
5. подключаемся через локальную машину к графическому интерфейсу RabbitMq 

ex 2
1. вводим логин и пароль guest guest - пользователь, созданный автоматически при установке RabbitMq
2. создаем очередь с именем Payment, остальные настройки по умолчанию - Durable и Auto delete - no
2. создаем очередь с именем Delivery, остальные настройки по умолчанию - Durable и Auto delete - no
2. создаем очередь с именем Feedback, остальные настройки по умолчанию - Durable и Auto delete - no

ex 3
Реализовать цепочку «Заказ еды — оплата — доставка — отзыв клиента»
Я бы реализовала скрипт, который при заказе еды получает список, заказанный клиентом, рассчитывает сумму заказа, доставку, применяет скидки и т.д.
Далее этот скрипт кладет сообщение с суммой для оплаты и контактами клиента в очередь Payment
Другой скрипт вычитывает сообщение из очереди, формирует счет для оплаты клиента и отправляет ему информацию для оплаты.
Клиент получает счет, оплачивает его и присылает подтверждение на email
Третий скрипт разбирает сообщения, которые пришли на почту с информацией об оплате и формирует сообщение для очереди Delivery с информацией о заказе - список заказанных продуктов и адресом доставки
Четвертый скрипт вычитывает сообщение из очереди и формирует оповещение для службы доставки (например, сообщение в телеграмм).
Служба доставки, когда доставка произведена, оповещает об этом систему (пусть через телеграмм бота с номером заказа)
Пятый скрипт получает уведомление от телеграмм бота о том, что доставка осуществлена, формирует сообщение с информацией о заказе для очереди Feedback и отправляет его
Шестой скрипт вычитывает сообщение из очереди Feedback, отправляет клиенту сообщение с просьбой оставить отзыв(давайте опять через телеграмм)
Седьмой скрипт получает отзыв клиента и сохраяет его в БД

Реализую запись одного сообщения в очередь и затем его вычитывание из этой очереди
